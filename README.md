# BloodLink_DonorMobileApp

A responsive Flutter donor app UI built with reusable widgets, clean navigation, and Material Design.

## Structure

- `lib/main.dart` — app entry point and bottom navigation
- `lib/screens/` — home, campaigns, urgent, history, and profile screens
- `lib/widgets/` — reusable `CustomButton`, `CustomTextField`, and `CustomCard`
- `lib/theme/` — shared colors and text styles

## Run

1. Install Flutter SDK.
2. Open the project folder.
3. Run `flutter pub get`.
4. Run `flutter run`.
